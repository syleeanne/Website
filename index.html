    [Ongoing 2019] Expressing the interface's personalities with color and motions

    The project aims to figure out the cues to express the personality of interfaces. Visual cues (color and motion) and verbal cues (voice pitch, emotionality, gender, speed, wordiness) were selected for modalities.

    [Ongoing 2018] Designing the social dialogue model for chatbot

    Using the automation server and Google API system, users type in the Google form and the conversation data is being accumulated in the context of counseling. The accumulated data will be analyzed and tagged with social cues, such as trustworthiness, intimacy, and self-disclosure using Python Markovify modeling.
